# The Snake App Web Page
## Description
TSA Landing page it's just a SPA for the users to know a little bit more of this app.

This page has some principal sections:

1. Welcome and sign up form (Hero)
2. Functions
3. Download buttons
4. Request species (Special feature in the main app)
5. links to social media
6. Copyright footer.
